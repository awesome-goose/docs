### Installation

To install Goose CLI, you can use the following command:

```bash
go install github.com/awesome-goose/cli@latest
```

Make sure your Go environment is set up correctly and that your `GOPATH/bin` is included in your system's `PATH` variable to run the `goose` command from anywhere in your terminal.

You can verify the installation by checking the version of Goose CLI:

```bash
goose --version
```

This should display the installed version of Goose CLI, confirming that the installation was successful.
