### API

#### Overview

Goose provides all you need to initiate, build and deploy your API application with ease.

`You need [Goose CLI](../cli/index.md) for this section`

#### Bootstrap

```
$ goose init --platform=api --name=my-app
```

#### Run

```
$ cd ./my-app
$ goose run
```
